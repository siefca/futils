futils overview

Function Utilities Library

1    Introduction

futils is a library that provides a set of forms that add some abstractions for managing functions in Clojure.

Currently implemented macros and functions are:

  • argc – counts arguments a function takes (for all arities),
  • relax – wraps a function in a way that it accepts any number of arguments,
  • args-relax – like frelax but it requires to explicitly describe the accepted arities,
  • frepeat – creates a sequence of results returned by the given function (which uses named arguments),
  • mapply – works like apply but for named arguments.

2    Installation

Add to project.clj dependencies:

[pl.randomseed/futils "0.5.0"]

Then require it in your program:

(require 'futils.core :as futils)

or:

(ns your-namespace (:require [futils.core :as futils]))

3    Usage

3.1    argc

about argc

"counts positional arguments" (argc (fn [x])) => (contains {:arities #{1} :engine keyword? :f ifn? :variadic false}) (argc (fn ([x y])([z]))) => (contains {:arities #{1 2} :variadic false})
"counts variadic arguments" (argc (fn [a & b])) => (contains {:arities #{2} :variadic true}) (argc (fn ([])([& a]))) => (contains {:arities #{0 1} :variadic true})
"works on named functions and Vars" (defn- fun ([]) ([a]) ([a b]) ([a b & c])) (argc fun) => (contains {:arities #{0 1 2 3} :variadic true}) (argc #'fun) => (contains {:arities #{0 1 2 3} :variadic true})
"works on macros" (defmacro mak ([]) ([a]) ([a b]) ([a b & c])) (argc #'mak) => (contains {:arities #{0 1 2 3} :macro true :variadic true})
"returns nil if the given argument is not a function" (def notfun) (argc 1) => nil (argc nil) => nil (argc "a") => nil (argc notfun) => nil (argc String) => nil
